Alerts, Toasts & Banners

Feedback components communicate important system state changes, errors, and informational messages to the user in a non-disruptive yet noticeable manner.

1. Toasts (Transient Feedback)

check_circle

Course Published

Students can now access the materials.

error

Upload Failed

File size exceeds the 50MB limit.

warning

Session Expiring

Please save your work soon.

Toasts appear temporarily at the bottom edge of the screen to provide concise feedback about an operation. They should stack if multiple arrive simultaneously.

  • Success: Green accents (`tertiary-fixed-dim`)
  • Error: Red accents (`error`)
  • Warning: Gold/Amber accents (`secondary`)
  • Info: Primary brand accents (`primary`)
